# Vela Samples

[Xiaomi Vela 快应用](https://iot.mi.com/vela/quickapp/zh/content/intro.html)官方示例项目


## 简介
本应用框架是一套以前端开发技术栈为主进行应用开发的框架，采用流行的前端开发模式，贴合主流前端开发者的思维习惯，同时大幅提升应用的性能，提供大量前端环境无法使用的系统能力，以及很多第三方服务的对接能力。

此项目为示例项目,帮助开发者快速入门
