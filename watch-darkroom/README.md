# 小黑屋 · Watch Dark Room

一款移植到 **小米 Vela 手表**（快应用 / `.ux`）上的放置文字冒险游戏，改编自经典网页游戏 [A Dark Room](https://adarkroom.doublespeakgames.com/)（MIT 许可）。纯原生快应用实现，未使用任何原版前端代码。

> 提示：原版 A Dark Room 在 iPhone / iPad / 安卓 / 电脑上都能玩，本项目是它的手表移植版本。

---

## 剧情

你在漆黑、寒冷中醒来，身边只有一座将熄的火堆。

从生火、添柴开始，温暖会吸引一位神秘的**建造者**前来，她能帮你打造陷阱与工具。渐渐地，流浪者循着火光聚拢，一个小小的聚居地在森林边缘成形——棚屋、猎人小屋、制革坊、炼钢厂依次立起，村庄日益喧闹。

但森林之外，是一片被遗忘的荒原。废城、矿井、沼泽、古老的战场散落其间，潜伏着野兽与劫掠者。在荒原最深处，一艘**坠毁的星舰**正等待着被重新唤醒。

收集、建造、远征、战斗——最终修复星舰，挣脱这片灰烬之地，飞向群星。

## 玩法

- **生火间**：生火 / 添柴维持火焰，保持温暖让建造者恢复并解锁建造。
- **建造**：陷阱、手推车、棚屋、猎人小屋、贸易站、制革坊、烟熏房、工坊、炼钢厂、军械库……
- **村庄**：棚屋吸引人口，把村民分配为伐木者 / 猎人 / 陷阱师 / 皮革师 / 熏肉师 / 矿工，每 10 秒自动产出。
- **制造**：工坊解锁武器护甲；荒野中的**外星制造机**解锁能量刃、等离子枪等高级装备。
- **贸易**：游牧商人用毛皮换取鳞片、牙齿、罗盘等稀有物资。
- **随机事件**：15 种事件以弹窗形式出现，需要你做出抉择。
- **荒野探索**：拿到罗盘后**出征补给**装备出发，ASCII 地图回合制移动，管理水与熏肉，遭遇战斗，搜刮 14 种地标。
- **太空小游戏**：修复星舰后躲避陨石爬升，撑到终点即通关。可用外星合金升级船体与引擎。
- **重生**：通关后可携声望重新开始。

## 操作

- 点击按钮执行操作，冷却中按钮变暗并读秒，资源不足同样变暗。
- 荒野中用方向键（北 / 南 / 西 / 东）移动，下滑可见进食与返回。
- 战斗中每把武器有独立冷却，远程消耗弹药，流星索可眩晕敌人。

### 开发者设置（仅供测试）

一键就绪建造者、资源大礼包、解锁全部建筑、解锁探索与装备、全部武器弹药、出征补给、解锁外星制造机、升级飞船、重生、触发任意事件 / 地标 / 战斗、测试太空小游戏、显示性能数据等。
